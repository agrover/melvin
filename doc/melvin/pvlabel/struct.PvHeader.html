A struct containing the values in the PV header. It contains pointers to the data area, and possibly metadata areas and bootloader area.

Fields

uuid

The unique identifier.

size

Size in bytes of the entire PV

ext_version

Extension version. If 1, we look for an extension header that may contain a reference to a bootloader area.

ext_flags

Extension flags, of which there are none.

data_areas

A list of the data areas.

metadata_areas

A list of the metadata areas.

bootloader_areas

A list of the bootloader areas.

dev_path

The device this pvheader is for
